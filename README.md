# OHC — One Hand Control
## Luxury Renovation & Interior Design Website

### Tech Stack
- React 18 + Vite
- Framer Motion (animations)
- React Icons
- Custom CSS variables (no Tailwind dependency)
- Google Fonts: Cormorant Garamond + Jost

### Setup

```bash
npm install
npm run dev       # Development server
npm run build     # Production build
```

### Features
- Animated image slider hero (Ken Burns effect, auto-play, fade transitions)
- Sticky transparent navbar with hamburger mobile menu
- About section with stats
- 12-service grid with hover effects
- Gallery with lightbox (click any image)
- Why Choose Us section
- 4-step process timeline
- Testimonials carousel
- Contact form
- WhatsApp floating button
- Back-to-top button
- Loading screen
